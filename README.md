# Static Site
## Based on Yeoman simple-static-site

Static HTML/CSS/JS site/app for projects or scratchsheet

Requirements:

  * [Node.js](http://nodejs.org/).

To set-up your environment execute:

    $ npm install

To build execute:

    $ npm run build

To test and dev

    $ npm run start

    opens http://localhost:8000 in your browser.

To clean build:

    $ npm run clean
